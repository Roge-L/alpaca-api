RobinGood - Alpaca API Endpoint

This is RobinGood's endpoint to Alpaca API, subsections below describe how to access functionalities. All return values are in JSON notation.

Currently a user's key and secret key must be used.

Placing an order

There exists five types of orders: market orders, limit orders, stop orders, stop limit orders, and trailing stop orders.

  • Place a market order by accessing: https://robingood-alpaca-api.herokuapp.com/market/symbol/qty/side/time_in_force

  • Place a limit order by accessing: https://robingood-alpaca-api.herokuapp.com/market/symbol/qty/side/time_in_force/limit_price

  • Place a stop order by accessing: https://robingood-alpaca-api.herokuapp.com/market/symbol/qty/side/time_in_force/stop_price

  • Place a stop limit order by accessing: https://robingood-alpaca-api.herokuapp.com/market/symbol/qty/side/time_in_force/limit_price/stop_price

  • Place a trailing stop order by accessing: https://robingood-alpaca-api.herokuapp.com/market/symbol/qty/side/time_in_force/trail_percentage

Account details

  • View account equity by accessing: https://robingood-alpaca-api.herokuapp.com//equity

  • View account buying power by accessing: https://robingood-alpaca-api.herokuapp.com/buying_power

  • View account orders by accessing: https://robingood-alpaca-api.herokuapp.com/orders

  • View account positions by accessing: https://robingood-alpaca-api.herokuapp.com/positions
